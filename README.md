# 🍃 Cotton Visuals

> A stunning visual-only Fabric mod for Minecraft 1.21.4 — no cheats, pure aesthetics.

## ✨ Features

- 🫧 **Liquid Glass GUI** — iOS-inspired blurred interface with shimmer effects
- 🎨 **3 Visual Styles** — Liquid Glass, Classic, and fully Custom
- ✦ **Jump Circles** — glowing rings on every jump
- 🌈 **Trails** — colorful particle trails as you move  
- 💎 **Hit Particles** — 12+ styles on every hit
- 🎇 **16 GUI Particle Types** — sparkle, fire, crystals, hearts and more
- 🎯 **Target HUD** — beautiful player info display
- 🌿 **Ambient Effects** — fireflies, petals, stars around you
- 🔊 **Custom Sounds** — satisfying UI clicks and kill sounds
- 📁 **Config System** — save and load your setups
- 🌍 **Russian & English** — full language support

## 🎮 Purely Cosmetic

Cotton Visuals adds **zero gameplay advantage** — everything is visual only.
Play beautifully, play fairly.
